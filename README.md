# Matplotlib Tutorial

## Overview

This repository contains a hands-on Jupyter Notebook introducing Matplotlib, one of Python's most popular libraries for data visualization. It demonstrates how to create and customize different types of plots using practical examples.

## Topics Covered

- Introduction to Matplotlib
- Line Plots
- Plot Formatting
- Titles and Axis Labels
- Multiple Line Plots
- Legends
- Gridlines
- Plot Styles
- Saving Figures
- Bar Charts
- Horizontal Bar Charts
- Scatter Plots
- Pie Charts
- Plot Customization

## Technologies Used

- Python
- Matplotlib
- NumPy
- Jupyter Notebook

## Installation

Install the required libraries:

```bash
pip install -r requirements.txt
```

## Requirements

- Python 3.x
- Matplotlib
- NumPy
- Jupyter Notebook

## Running the Notebook

```bash
jupyter notebook
```

Open `Matplotlib_Tutorial.ipynb` and execute the cells sequentially.

After completing this notebook, you will be able to:
